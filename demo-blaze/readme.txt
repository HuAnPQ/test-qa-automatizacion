A continuación se detalla las instrucciones de ejecución del proyecto java con Maven para el ejercicio de API DemoBlaze:

1. Clonar el repositorio

2. Abrir el archivo de prueba login.feature o signup.feature.

3. Ejecutar la prueba del escenario, en caso de desear ejecutar todas los escenarios usar el archivo src/test/java/ParallelTests.java.

4. Validar los resultados del escenario y sus pasos, esto de el reporte de KarateLabs (target/karate-reports/karate-summary.html) o Cucumber (build/cucumber-html-reports/overview-features.html).
